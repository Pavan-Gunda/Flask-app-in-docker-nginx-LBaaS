# Flask-app-in-docker-nginx-LBaaS

a flask based app, load balanced using nginx.  


![img-1](flask.PNG)

## installation 

install docker and docker-compose. 


## Usage
clone the repository and run the following command from within the directory 
```python
docker-compose up -d 
```
